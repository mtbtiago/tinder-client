Tinder.js

HTML5 APIs in Practice

Tinder.js is a set of javascript files that encapsulate HTML5 API examples. Click the buttons in order and wait to receive the alerts confirming that all worked.

Geolocation

Retrieves the browser geolocation with the Geolocation API and displays coordinates.

AJAX

Uses the XMLHttpRequest API to retrieve and display an address using the Google Maps Static API.

Local Storage

Saves the coordinates, map and address into local storage as an object, using the LocalStorage API.

Notification

Retrieves the saved data and displays it in a notification using the Notification API with the Google Maps Static API map.
